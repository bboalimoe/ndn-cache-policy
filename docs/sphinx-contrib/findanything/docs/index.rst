=============================
FindAnything Sphinx Extension
=============================

Motivated by great findanything feature of `Sublime Text Editor`_, I have
written this extension.  For see it in action, simply start to type, to
find a topic or an index entry.

This extension is basically a wrapper and configuration of `jQuery FuzzyFind`_
plugin.
