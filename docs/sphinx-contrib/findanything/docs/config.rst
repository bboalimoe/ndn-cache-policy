Configuration
=============

.. confval:: html_findanything_add_indexentries

   Default is **True**, define if you want to add indexentries to 
   findanything list.

.. confval:: html_findanything_add_topics

   Default is **True**, define if you want to add topics to findanything 
   list.

.. confval:: html_findanything_css

   Default is ``findanything.css``.

.. confval:: html_findanything_hit_cacher

   Default is ``cookie``, and there is no other value supported right now.

.. confval:: html_findanything_use_cached_hits

   Default is **True**, set if there shall be used cached hits (which have
   been selected by user) to boost these values in next searches.
